Utilizing The Weather Channel API

In this example I've created a simple Angular.js app which submits a user's zip code via http request to The Weather Channel's free online API. This returns a JSON object with a ton of useful information we can access and repurpose for use in maps layers or weather forecasting, including temperature, precipitation, wind, even latitude and longitude. The raw output is below.



{{ city }}, {{ state }}

Weather Channel API geolookup response:

      {{ georesponse | json }}
